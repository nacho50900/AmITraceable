[[section-context-and-scope]]
== Contexto y alcance

=== Contexto de negocio

[cols="1,2,2", options="header"]
|===
| Interlocutor | Entradas a AmITraceable | Salidas de AmITraceable

| Usuario final
| Consentimiento OAuth, ninguna otra entrada manual (no hay formularios
  de datos personales)
| Informe de exposición (score, atributos inferidos, recomendaciones)

| Reddit (Data API)
| `access_token` tras autorización del usuario
| Peticiones de lectura: perfil, posts enviados, comentarios

| Instagram (Platform API / Business Login)
| `access_token` de corta y larga duración tras autorización
| Peticiones de lectura: perfil básico, media del usuario

| X / Twitter
| _(sin integrar; tarjeta "Coming Soon" en la interfaz)_
| —

| SonarCloud
| Resultados de análisis estático + cobertura (`lcov.info`,
  `coverage.xml`) enviados desde el pipeline de CI
| Quality gate, métricas de calidad de código

| GitHub Actions (CI/CD)
| Push / Pull Request al repositorio
| Ejecución de tests, build, análisis Sonar, publicación de imágenes
  Docker en releases
|===

=== Contexto técnico

[source]
----
                     HTTPS                         HTTPS (OAuth + REST)
 [ Navegador ] <----------------> [ webapp (React SPA, Nginx) ]
       |                                        |
       |                                        | HTTPS/JSON + cookie de sesión firmada
       |                                        v
       |                          [ users (FastAPI, Python) ]
       |                                        |
       |                     +------------------+------------------+
       |                     |                                     |
       |                 HTTPS OAuth/REST                    HTTPS OAuth/REST
       |                     v                                     v
       |            [ Reddit Data API ]                 [ Instagram Platform API ]
       |
       |  /metrics (Prometheus scrape, red interna Docker)
       v
[ Prometheus ] ---- datasource ----> [ Grafana ]
----

Mapeo de canales:

[cols="1,2,2", options="header"]
|===
| Canal | Protocolo | Datos que transporta

| Navegador <-> webapp
| HTTPS
| HTML/JS/CSS estático (SPA), peticiones `fetch` a la API del backend

| webapp <-> users
| HTTPS + JSON, cookie de sesión firmada (`SessionMiddleware`)
| Estado de autenticación por plataforma, informe de exposición

| users <-> Reddit / Instagram
| HTTPS (OAuth 2.0 + REST/JSON)
| Intercambio de `code` por `access_token`, lectura de perfil y
  contenido público

| Prometheus <-> users
| HTTP (scrape interno, red Docker `monitor-net`)
| Métricas de la instrumentación FastAPI (`/metrics`)

| Grafana <-> Prometheus
| HTTP (datasource)
| Consultas de métricas para los dashboards
|===
